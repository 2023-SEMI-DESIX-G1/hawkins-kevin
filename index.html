¿Qué es JavaScript y en qué se asemeja a Java?

JavaScript es un lenguaje de programación el cual nos permite crear contenidos dinámicos e implementar funciones complejas en páginas web. JavaScript se caracteriza por ser ligero y por permitir crear interacciones más dinámicas al desarrollar páginas web, aplicaciones, servidores e incluso juegos.

Java y JavaScript a pesar de ser lenguajes diferentes, poseen ciertas similitudes entre ellas se pueden mencionar que en ambos lenguajes se puede realizar la declaración de variables utilizando la palabra clave ra o let, en ambos lenguajes se pueden usar las condicionales if y ese para contralar los flujos de ejecución del programa, en ambos lenguajes se pueden definir funciones para agrupar bloques de código y en ambos lenguajes se pueden utilizar los bucles for y while para repetir bloques de código.
